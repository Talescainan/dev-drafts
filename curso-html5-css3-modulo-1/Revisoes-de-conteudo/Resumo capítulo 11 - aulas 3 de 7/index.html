Aula 3 - Inserção de áudios

Para inserir um áudio em uma página HTML, podemos usar a tag <audio>. Esta tag permite que você incorpore arquivos de áudio em sua página da web. Você pode especificar o arquivo de áudio usando o atributo src, e também pode fornecer controles para os usuários interagirem com o áudio usando o atributo controls.
