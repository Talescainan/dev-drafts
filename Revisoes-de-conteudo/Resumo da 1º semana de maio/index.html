Citações, códigos e Listas HTML

Citações

As citações são usadas para destacar um trecho de texto. Existem dois tipos principais de citações em HTML:

  • <blockquote>: Usado para citar um bloco de texto, geralmente com recuos.
  • <q>: Usado para citações curtas dentro de um parágrafo.

Códigos

Para exibir código em HTML, você pode usar as seguintes tags:

  • <code>: Usado para marcar um trecho de código inline.
  • <pre>: Usado para preservar a formatação do código, como quebras de linha e espaços.

Listas

Existem três tipos principais de listas em HTML:

  • <ul>: Lista não ordenada, onde os itens são marcados com bullets.
  • <ol>: Lista ordenada, onde os itens são numerados.
  • <dl>: Lista de definição, usada para pares de termos e descrições.
