Aulas 03/04 - PSEUDO CLASSES

Aula 1: Identificadores em CSS

O que é?

Identificadores em CSS são usados para selecionar elementos HTML específicos para aplicar estilos. Eles são definidos usando o atributo "id" em um elemento HTML e são referenciados no CSS com o símbolo "#".

Exemplo de uso

HTML:

                
                    <div id="meu-identificador">Conteúdo do div</div>

CSS:

                
                    <style>
                        #meu-identificador {
                            color: red;
                            font-size: 20px;
                        }
                    </style>

Aula 02: As diferenças entre id e class

Um identificador tem como principal função, destacar somente um unico elemento dentro do HTML. O que torna uma classe mais acessível, pois não há limites de uso, possibilitando variás elementos com classes diferentes, sem a limitação de somente uma. Um identificador pode até ser usado variás vezes, mas utilize nas tags principais para uma melhor organização.

Aula 03: Pseudo-classes em CSS

Pseudos-classes são literalmente, ":", elas tem que está relacionada a um elemento ou uma classe.

Exemplo de uso:

                
                <style>
                    div:hover > p {
                        display: block;
                        color: white;
                        background-color: red;
                        width: 300px;
                    }
                    div:hover {
                        color:red
                    }
                </style>
